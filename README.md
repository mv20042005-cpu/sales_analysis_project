# 📊 Retail Sales Data Analysis using Python

## 📌 Project Overview
This project focuses on analyzing retail sales data using Python to extract meaningful business insights. The analysis includes data cleaning, preprocessing, trend analysis, and data visualization to understand sales performance across different categories, regions, and time periods.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Calculate total sales and profit
- Analyze monthly sales trends
- Identify top-performing products and categories
- Generate visual insights using charts

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

---

---

## 📊 Key Features
- Data cleaning and preprocessing using Pandas
- Feature engineering (Sales = Quantity × Unit Price)
- Monthly sales trend analysis
- Category-wise sales analysis
- Top 5 products identification
- Region-wise performance analysis
- Data visualization using Matplotlib & Seaborn

---

## 📈 Key Insights
- 📅 March 2025 recorded the highest sales performance
- 📉 Sales dropped in April indicating a possible seasonal trend
- 🖥 Electronics category generated the highest revenue
- 🛋 High-value products contributed significantly to total sales
- 🌍 Different regions showed varying sales performance patterns
